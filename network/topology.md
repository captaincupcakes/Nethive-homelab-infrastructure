# Network topology

## Hardware
- Router: Netgear Nighthawk RAX50
- Switch 1 (upstairs): TP-Link TL-SG108
- Switch 2 (downstairs): TP-Link TL-SG108

## Layout

- Switch 1 feeds Xerox Workcentre 6605DN and Zebra GK420D
- Switch 2 feeds Nobara Rig, Test Mini PC 1, Windows Mini PC 2
- Eos-Laptop connects via WiFi (poor signal, extended antennas)
